# Perkkk | Creative Developer Portfolio 🟢

> "Building the future, one commit at a time."

This is the source code for **[perkkk.dev](https://perkkk.dev)**, my personal portfolio showcasing my journey as a Computer Science student and Creative Developer. The site features a cinematic, terminal-inspired aesthetic with high-performance animations.

## 🚀 Status: Launching Soon (27.03.2026)

Currently serving a "Coming Soon" landing page with a custom WebGL-based letter glitch effect.

## 🛠 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: 
  - Custom Canvas/WebGL (`LetterGlitch`)
  - `tailwindcss-animate`
  - `lucide-react` for icons
- **Font**: Geist Mono & Sans

## ✨ Features

- 🎥 **Cinematic Background**: A custom implementation of a matrix-like letter glitch effect.
- ⚡ **High Performance**: Optimized canvas rendering for the background.
- 🎨 **Glassmorphism UI**: Modern, clean, and tech-focused interface.
- 📱 **Fully Responsive**: seamless experience across all devices.
- 🔍 **SEO Optimized**: Metadata configurations for social sharing.

## 💻 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
